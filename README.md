# Advanced Encryption Tool (AES-256)

## Description
A secure file encryption and decryption tool using AES-256 encryption with a user-friendly GUI.

## Features
- AES-256 Encryption
- Password-based key generation
- Encrypt & Decrypt any file
- Simple GUI (Tkinter)

## Technologies
- Python
- Cryptography Library
- Tkinter

## How to Run
pip install cryptography  
python encryption_tool.py
